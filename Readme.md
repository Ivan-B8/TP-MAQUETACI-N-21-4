Luz de Preta

Proyecto web desarrollado con HTML y CSS que presenta diferentes secciones visuales, galería de imágenes y páginas informativas.


Descripción

Luz de Preta es un sitio web que incluye:

Página principal
Galería de imágenes
Sección de cards
Página de contacto

El objetivo de este proyecto es practicar estructura HTML, estilos CSS e implementar la organización de carpetas.

Tecnologías utilizadas

HTML5
CSS3
